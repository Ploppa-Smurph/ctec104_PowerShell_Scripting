Some Useful Methods of Strings

ToUpper() - converts all letters in the string to uppercase

ToLower() - converts to lowercase

Contains("val") - returns true or false

Substring(x,y) - returns value starting at x for y characters (0 indexed)

Substring(x) - returns value of x to the end 
    $s="Hello"
    $t=s.ToUpper()    -- takes Hello and makes it all uppercase
    $s.Contains("L")   -- False -- $s only contains lowercase 'l'
    $s.Contains("H")   -- True $s does contain 'H'
    $u = $s.Substring(1,3)  -- will store the characters of the string $s from 'e'(index 1), to the 2nd 'l'(index 3). 
            $u - will contain 'ell'. characters starting at index 1 and ending at 3

Splitting and Joining Strings

PS C:\Users\JSWalley> $m = "str1 str2 str3 str4"
PS C:\Users\JSWalley> $n = $m.Split()
PS C:\Users\JSWalley> $n
str1
str2
str3
str4
PS C:\Users\JSWalley> $n.ToString()
System.String[]

there is also a Join command to do the opposite of split. you can specify whether or not to place separator between the strings. 
    $p = -join $n                -- will join with NO space   
    $q = $n -join " "            -- will join with a space
    $r = $n -join ", "           -- will join with a comma and space following
get-help about_Split
get-help about_Join -online -- to read more
