this chapter will cover: File System Basics, Standard Commands, File Properties, File Permissions.

File System - hierarchical namespace containing the directories and files on your computer. includes all system drives where files are stored.
includes HD, external drives, USB, and DVD drives.

C: is normally the OS hard disk. A: and B: are rarely used any longer because they are associated with the floppy drives of yore.
the top level directory is identified with a backslash '\' ex: top level directory of C drive is C:\.
Any directory created inside the top level directory is a child or subdirectory of ie: C:\, the child directories of a child is also a subdirectory.
Files are then stored in the directories which creates the orginization system for file storage.

'.' = current directory; '..' = parent directory

    Common CLI commands that are PowerShell aliases
  • cd - change directory - Set-Location
  • copy - copy a file - = cp - Copy-Item
  • del - delete a file or directory - Remove-Item
  • dir - directory listing [does the same as 'ls'] - Get-ChildItem
  • mkdir - make a new directory
  • move - move a file - = mv - Move-Item
  • ren - rename a file - Rename-Item
  • rmdir - remove a directory
  • cat - Get-Content
  • gp - Get-ItemProperty
  • pwd - Get-Location

change the current drive by: entering "cd d:" -- also you could simply enter "d:"" at the prompt to change the current drive.

to move a file and rename at the same time: "move file1.txt C:\dir2\file2.txt" -- renames the file while it moves the file.

File Properties

when you ls or dir the output lists all of the files and directories in the current directory.
this ouptut contains 4 properties: Mode, Last Write Time, Length, and Name

files and directories have many more properties that are NOT listed by the dir command.
-- to get all the properties of a file or directory:
"get-item filename | select *" or "dir | select *" -- the 2nd will display of every file in the directory. get-item * | select * does the exact same thing

select certain properties to display: dir | select name, length, attributes

file permissions define the access rights that user's have when accessing the files and directories.
the 'Get-Acl' command is used to get limited file permissions: get-acl filename
to get the full permissions type: get-acl filename | select *

the most important permissions we will concentrate on are: AccessToString, Owner, and Group
AccessToString displays what control you have over a file

ACL stands for Access Control Log.
