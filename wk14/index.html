PERL - Practical Extraction and Report Language

PERL is a scripting language designed to manipulate Text. PERL borrows syntax and context from many langauges
PERL has an extensive ability to interface with multiple resources:
Database integration supports Oracle, Sybase, Postgres, MySQL and others
works with HTML, XML, and other mark-up languages. Often imbeded in web pages to speed up text processing

Variables

in Perl work as they do in other languages. PERL supports 3 types of variables:
  • Scalar - store single value that can be number, string, or reference
  • Array - store multiple values
  • Hashes - store key / value pairs
Perl supports all standard math operators +-*/% and ** for exponents
String variables are the same as in other languages.
Reference variables hold the location of variables (like Pointers in C)
Arrays hold multiple values (numeric, string, or reference)

@vals = (25, 30, 35) is an ex: of an array in Perl. Arrays in Perl start with '@'
ex:$v = $vals[1] will set the value '30' to the variable '$v'

Hash - a hash is a set of key/value pairs that can be stored in any order
% precedes a hash group. ex: $data{'a' = 45}; $data{'b' = 50} adds data 1 pair at a time;
%data = ('a', 45, 'b', 50, 'c', 55) -- enters grouped data into a hash array

you can extract all the keys from a hash: @k = keys %data
extract the value from a hash: @a = values %data

Control Structures

Perl supports standard control structures (if / if-else; while; for; and foreach)
Perl uses the same syntax as C
  • if if (condition) { }
  • if-else if (condition) { } else { }
  • while while (condition) { }
  • for for ($i = 1; $i <=1 0; $i=$ i +1) { }
  • foreach ex: @list = (14, 17, 22, 26, 30, 33, 39)
    • foreach $a (@list) { } - will iterate over the list

PHP

PHP allows webpages to interact with databases. PHP supports MySQL, PostgreSQL, Oracle, Sybase, MicrosoftSQL and more
PHP is similar to C. PHP is imbedded in HTML like JavaScript and VBScript.
ALL PHP code is processed on the web server and output to the client is pure HTML
    3 components are required to use PHP:
  • a web server like Apache or Microsoft IIS
  • a database
  • a PHP parser to generate HTML output

variables in PHP begin with $
variables can be integers, doubles, strings, Booleans, and references
in PHP variables are not assigned a data type -- PHP resolves the data type based on how the variable is used
PHP supports all standard math operators, and also includes '++'' and '--'' support to increment/decrement
    PHP supports 3 variants of Array
  • Standard array = $arr = array (1, 2, 3, 4, 5)
  • Associative array has key/value pairs = $arr = array('a' => 10, 'b' => 20)
  • Two-dimensional array an array where each element is a subarray
